# 滤波器构造 (Gabor-like) def _construct_filters(self): t = linspace(-1, 1, kernel_size=7) for i in range(4): center_f = sigmoid(center_freqs[i]) * π bandwidth = sigmoid(bandwidths[i]) * 0.5 + 0.1 modulation = cos(2π * center_f * t) envelope = exp(-t² / (2 * bandwidth²)) filter[i] = base_filters[i] * modulation * envelope

损失函数组成
id_criterion.py
▼
总损失公式
L_total = L_detr + λ₁ * L_id + λ₂ * L_ortho + λ₃ * L_consistency # 默认权重 λ₁ = 1.0 # ID损失权重 λ₂ = 0.1 # 正交性损失权重 λ₃ = 0.05 # 一致性损失权重
各损失详解
L_detr DINO检测损失
loss_ce分类损失 (Focal Loss)
loss_bboxL1边界框损失
loss_giouGIoU损失
权重ce=2.0, bbox=5.0, giou=2.0
L_id ID分类损失
类型CrossEntropyLoss
类别数51 (50 ID + 1 newborn)
辅助损失6层解码器各层输出
L_ortho 频带正交性损失
公式||C - I||² (C为频带相关矩阵)
目的确保不同频带捕获不同信息
L_consistency 双分支一致性损失
公式KL(p_std || p_freq) + KL(p_freq || p_std)
目的鼓励两分支预测一致
