## 6. 风险与缓解

- [ ] **训练不收敛**：降低LR，冻结DETR，检查梯度  
- [ ] **显存不足**：减小batch或频带数，关闭辅助loss  
- [ ] **结果不稳定**：多次运行取平均，固定种子  
- [ ] **审稿质疑创新性**：强调频率视角与遮挡鲁棒性实验
