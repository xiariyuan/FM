# Pro Review Interaction Entry Template

每次与 Pro 聊天结束后，立即复制这段模板到 `md/PRO_REVIEW_INTERACTION_LOG.md` 末尾并填写。

```markdown
## YYYY-MM-DD

### 主题

[一句话概括本次提问]

### 对应提示词

- `md/...`

### 提问摘要

- [这次发给 Pro 的问题 1]
- [这次发给 Pro 的问题 2]
- [如果有明确候选集，也写清楚]

### Pro 回答摘要

- [Pro 回答的主结论]
- [Pro 给出的唯一主方案]
- [Pro 给出的备选或限制条件]

### Pro 核心回答

- 判定: `GO / NARROW GO / KILL`
- [核心判断 1]
- [核心判断 2]
- [核心判断 3]

### 我们最终采纳的动作

- [采纳了什么]
- [没采纳什么]
- [最终落到了哪个代码/实验动作]

### 后续证据修正

- [后续如果尚无新证据，先写 `暂无`]

### 相关文件

- prompt: `md/...`
- raw_answer: `[如有单独保存则写路径，没有可省略]`
- run_root: `[如有对应实验目录则写路径，没有可省略]`
```
